# Falling Stairs and Slabs

## Dependancies

Uses one of (in this preference order)

- `moreblocks`
- `mcl_stairs`
- `stairs`

To create slabs and stairs for each falling block.

- `default` for
  - `default:gravel`
  - `default:sand`
  - `default:desert_sand`
  - `default:silver_sand`
- `mcl_core`
  - `mcl_core:gravel`
  - `mcl_core:sand`
  - `mcl_core:redsand`
